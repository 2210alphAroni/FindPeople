## 🛠️ Tech Stack

**Frontend**

- React 18 + Vite
- React Router DOM
- Leaflet / React-Leaflet (Map)
- Socket.io Client
- Axios
- CSS Modules

**Backend**

- Node.js + Express
- Socket.io (Real-time)
- MongoDB Atlas + Mongoose
- JWT Authentication
- Multer (File Upload)
- Bcrypt (Password Hashing)

## 🚀 Getting Started

### Prerequisites

- Node.js v18+
- MongoDB Atlas account (free tier works)
- npm or yarn

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/findpeople.git
cd findpeople
```

### 2. Setup Backend

```bash
cd backend
cp .env.example .env
```

Edit `.env` file:

```env
PORT=5000
MONGODB_URI=mongodb+srv://<username>:<password>@cluster0.xxxxx.mongodb.net/findpeople?retryWrites=true&w=majority
JWT_SECRET=your_super_secret_key_here
CLIENT_URL=http://localhost:5173
```

```bash
npm install
npm run dev
```

### 3. Setup Frontend

```bash
cd frontend
npm install
npm run dev
```

### 4. Open in browser

```
http://localhost:5173
```

## 🛡️ Becoming Admin

After registering and logging in, open browser console (F12) and run:

```javascript
// Step 1: Get your token
localStorage.getItem("fp_token");

// Step 2: In terminal, run:
// curl -X POST http://localhost:5000/api/auth/make-admin \
//   -H "Authorization: Bearer YOUR_TOKEN_HERE"
```

Then refresh the page — **🛡️ Admin** button will appear in navbar.

## 🌐 API Endpoints

### Auth

| Method | Endpoint                     | Description           |
| ------ | ---------------------------- | --------------------- |
| POST   | `/api/auth/register`         | Register new user     |
| POST   | `/api/auth/login`            | Login                 |
| GET    | `/api/auth/me`               | Get current user      |
| PUT    | `/api/auth/change-password`  | Change password       |
| PUT    | `/api/auth/toggle-anonymous` | Toggle anonymous mode |
| POST   | `/api/auth/make-admin`       | Make first admin      |

### Users

| Method | Endpoint             | Description    |
| ------ | -------------------- | -------------- |
| PUT    | `/api/users/profile` | Update profile |
| POST   | `/api/users/avatar`  | Upload avatar  |

### Messages

| Method | Endpoint                | Description      |
| ------ | ----------------------- | ---------------- |
| GET    | `/api/messages/:userId` | Get conversation |
| POST   | `/api/messages`         | Send message     |
| POST   | `/api/messages/upload`  | Send file/photo  |

### Groups

| Method | Endpoint                   | Description          |
| ------ | -------------------------- | -------------------- |
| GET    | `/api/groups`              | Get my groups        |
| POST   | `/api/groups`              | Create group         |
| GET    | `/api/groups/:id/messages` | Get group messages   |
| POST   | `/api/groups/:id/messages` | Send group message   |
| POST   | `/api/groups/:id/upload`   | Upload file to group |
| DELETE | `/api/groups/:id/leave`    | Leave group          |

### Admin

| Method | Endpoint                    | Description      |
| ------ | --------------------------- | ---------------- |
| GET    | `/api/admin/stats`          | Get statistics   |
| GET    | `/api/admin/users`          | List all users   |
| DELETE | `/api/admin/users/:id`      | Delete user      |
| PUT    | `/api/admin/users/:id/ban`  | Ban/unban user   |
| PUT    | `/api/admin/users/:id/role` | Change user role |

## ⚙️ Socket Events

| Event             | Direction       | Description          |
| ----------------- | --------------- | -------------------- |
| `user:join`       | Client → Server | Join with user info  |
| `location:update` | Client → Server | Update GPS location  |
| `nearby:users`    | Server → Client | List of nearby users |
| `message:send`    | Client → Server | Send private message |
| `message:receive` | Server → Client | Receive message      |
| `typing:start`    | Client → Server | Typing indicator on  |
| `typing:stop`     | Client → Server | Typing indicator off |
